# DurabilityWarn
![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21.x-blue)
![License](https://img.shields.io/badge/License-MIT-blue)

**Stay safe! A [Meteor Client](https://github.com/meteor-development/meteor-client) addon that warns you when armor, elytra and tools are about to break.**

* Warnings appear in the actionbar.
* Ability to change the threshold and monitored slots in the Meteor GUI.
* Works with Meteor Client and forks like Asteroid.

## Installation
1. Download jar from [Releases](https://github.com/nivalunaa/durability-warn/releases).
2. Place in the `mods` folder.

## Build
Linux/Mac:
```bash
./gradlew build
```
Windows:
```cmd
gradlew build
```
Finished jar will be in build/libs/
